## ðŸŽ® Gameplay Mechanics
The game balances traditional platforming with "Educational Gatekeeping":

* **The Labyrinth Trials:** Each section of the manor is a mini-labyrinth. Reaching the end of a corridor brings you to a Seal.
* **The MCQ Seals:** To unlock a door or bridge a gap, you must interact with a "Memory Altar." This triggers an EFL challenge.
* **Visual Feedback:** Correct answers restore light and color to the room, while incorrect answers keep the path shrouded in shadow, forcing the player to reconsider their choice.
* **Memory Recovery:** Every successful lock opened restores a fragment of the story, giving the player a narrative reward for their grammatical accuracy.

## ðŸ“š Educational Impact
*Echoes of the Abyss* moves beyond worksheets by placing grammar in a high-stakes fantasy context:
* **Mastering Prepositions:** Specifically targets prepositions of place and time (in, on, at, behind, through).
* **Grammar in Action:** Focuses on Tense Consistency and Sentence Structure through contextual MCQ prompts.
* **Accessible Learning:** Simple controls ensure that students focus 10% on the platforming and 90% on language processing.

## ðŸš€ Installation & Setup
To run this project from the source code, follow these standard Unity procedures:

1.  **Clone the Repository:**
    ```bash
    git clone [https://github.com/YOUR_USERNAME/echoes-of-the-abyss.git](https://github.com/YOUR_USERNAME/echoes-of-the-abyss.git)
    ```
2.  **Unity Version:** Ensure you have **Unity 2022.3 LTS** installed via Unity Hub.
3.  **Opening the Project:**
    * Open **Unity Hub**.
    * Click **Add** > **Add project from disk**.
    * Select the `echoes-of-the-abyss-v0.8` folder.
4.  **First Run:**
    * Allow Unity to import assets and generate the `Library` folder (this may take a few minutes).
    * Locate the `Scenes` folder in the Project window.
    * Open `MainMenu.unity` and press **Play**.
